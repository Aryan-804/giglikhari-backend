# GigLikhari

Free AI tool for freelancers to generate Fiverr proposals, Upwork cover letters, gig titles, and bios in seconds.

**Live Demo:** https://giglikhari.pages.dev

### Features
- Proposal Writer – client-focused, 120-180 words
- Gig Creator – SEO titles, tags, pricing
- Bio Writer – Instagram, LinkedIn, Fiverr, Upwork
- No login, no payment, works instantly

### Built With
- Frontend: Cloudflare Pages
- Backend: Python Flask on Hugging Face Spaces
- AI: Google Gemini 2.0 Flash Lite

### Why I built it
I got tired of rewriting the same proposals. This tool does it in 10 seconds.
